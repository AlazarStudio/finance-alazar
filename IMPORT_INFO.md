# Информация об импорте данных

## Импортировано из Python программы:

- **Клиентов**: 13
- **Сотрудников**: 10
- **Постоянных расходов**: 11
- **Доходов**: 40

## Преобразование данных:

### Сотрудники
- `name` → `fullName`
- `position`, `percent` сохранены
- `salary` не используется в новой версии

### Клиенты
- `name`, `company`, `contact` сохранены
- Добавлены поля `phone`, `email` (пустые)

### Постоянные расходы
- `title` → `name`
- `amount` сохранен
- `period` оставлен пустым

### Доходы
- Преобразованы из структуры по месяцам в плоский массив
- Дата установлена как первое число месяца (можно улучшить, если есть точные даты)
- Для доходов с несколькими исполнителями:
  - `employeeId` берется от первого исполнителя
  - `employeePayouts` содержит общую сумму всех выплат
  - `employeePayoutType` определяется по типу первого исполнителя
- Все расчеты (налог, прибыль) пересчитаны

## Файл данных

Данные сохранены в: `server/data/data.json`

При запуске сервера эти данные будут автоматически загружены.
